# Precify

Precificador para Mercado Livre e Shopee.

## Subir no GitHub Pages (5 min)

1. github.com → New repository → `precify` → Public → Create
2. Upload: `index.html`, `README.md`, `supabase_schema.sql`  
3. Settings → Pages → Branch: main → Save
4. Link: `https://SEU_USUARIO.github.io/precify`

## Conectar Supabase

1. supabase.com → New project → aguarde
2. SQL Editor → New query → cole `supabase_schema.sql` → Run
3. Settings → API → copie **Project URL** e **anon public key**
4. No app: aba **Configurações** → cole URL e chave → **Salvar e conectar**

> As chaves ficam no seu navegador (não no GitHub). Cada dispositivo precisa configurar uma vez.

## Instalar no celular

**iPhone:** Safari → compartilhar → Adicionar à Tela de Início  
**Android:** Chrome → menu (⋮) → Adicionar à tela inicial
